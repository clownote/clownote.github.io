Rust (3)自定义类型

学习 Rust By Example 3. Custom Types

Rust 自定义类型有结构体（struct）和枚举（enum）。

结构体

Rust 的结构体有三种：

  • 单元结构体：没有字段的

  • 元组结构体：一个有名字的元组，一个字段匿名的结构体

  • 正常的 C 语言风格的结构体

注意元组结构体用圆括号 ()，有字段命名的结构体用花括号 {}。
